# Simple Rest App 

A simple Spring Boot application useful for experimenting with Open Shift.

## Running
```
mvn install  
mvn spring-boot:run  
```

## Endpoints
GET http://localhost:8090/greetings/{id}  
GET http://localhost:8090/greetings/random  
POST GET http://localhost:8090/greetings/  
	greeting body
